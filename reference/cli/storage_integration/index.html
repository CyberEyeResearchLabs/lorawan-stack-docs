Application Storage Integration CLI Commands

Command ttn-lw-cli applications storage get

Description

Retrieve stored upstream messages

Usage

ttn-lw-cli applications storage get [application-id] [flags]

Flags for ttn-lw-cli applications storage get

--after

type: string

query upstream messages after specified timestamp (format: '2006-01-02 15:04:05')

--after-utc

type: string

query upstream messages after specified timestamp (format: '2006-01-02 15:04:05') (UTC)

--application-id

type: string

--before

type: string

query upstream messages before specified timestamp (format: '2006-01-02 15:04:05')

--before-utc

type: string

query upstream messages before specified timestamp (format: '2006-01-02 15:04:05') (UTC)
